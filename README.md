# iPhone-13-Pro

## Описание

Одностраничный промо лэндинг Iphone 13 Pro с динамически подгружаемым контентом

## Функционал

- Адаптивная верска
- Плавный скролл
- Табы с подгружаемым контентом
- Плавный аккордеон
- Динамически подгружаемый контент с товарами
- Отправка формы на сервер

## Стэк технологий

- Native JS
- Fetch
- Инкапсуляция
